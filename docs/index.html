ROW - Rest Over Websocket

Row (aka Rest Over Websocket) makes your pre-ready rest api's accessible through websocket.

See project on github

Not much extra coding needed: With easy configuration, create websocket endpoint for your springboot app project.

Spring boot starter

Setting up the project could not be easier! Add springboot-row-starter package to you dependencies, and you have all the classes and configurations ready.

More Info

Rich annotation support

ROW mostly relies on java annotations. It supports most of the annotations that you are probably using in your rest controllers already. So there is not much you need to change in your code.

More Info

Configurable

Need to make change to ROW configurations? Simply inject your alternative beans and you are good to go!
